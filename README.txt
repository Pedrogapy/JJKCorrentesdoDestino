ARQUIVOS DE KIYOMI — SITE ESTÁTICO

Como usar:
1. Envie todos os arquivos deste ZIP para um repositório no GitHub.
2. No GitHub, vá em Settings > Pages.
3. Em Build and deployment, selecione a branch main e a pasta /root.
4. Salve e aguarde o link do GitHub Pages.

Páginas incluídas:
- index.html
- casos/escadaria-sem-numero.html
- casos/quarto-0b.html
- casos/viela-do-sino-mudo.html
- publico/professores.html
- publico/instituicao.html

O site não usa imagens externas nem backend.
